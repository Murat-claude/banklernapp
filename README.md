# 🏦 Bank-Lernapp für Schularbeit Vorbereitung

## ✨ Features
- 📚 5 Lernfelder (LF 1-5) mit allen Themen
- 🧪 Interaktive Quizzes mit Instant-Feedback
- 🏆 Prüfungs-Simulationen (90 Min & Mini-Tests)
- 📖 Umfassendes Bank-Glossar
- ✅ Täglicher Lernplan (Dienstag-Freitag)
- 📊 Fortschritts-Dashboard
- 🌙 Dark Mode
- 📱 Vollständig responsiv (Handy, Tablet, PC)
- ⚡ Offline-fähig
- 💾 Lokaler Speicher für Fortschritt

## 🚀 Auf Netlify Deployen (EINFACH!)

### Schritt 1: Dateien vorbereiten
```
banklernapp/
├── index.html          (Hauptdatei)
├── manifest.json       (PWA Config)
└── README.md          (Diese Datei)
```

### Schritt 2: GitHub Repo erstellen
1. Gehe zu **github.com**
2. Neue Repository: **"banklernapp"**
3. Alle 3 Dateien hochladen
4. **Commit**

### Schritt 3: Mit Netlify verbinden
1. Gehe zu **netlify.com**
2. **"New site from Git"**
3. GitHub auswählen & autorisieren
4. **banklernapp** Repository wählen
5. Deploy Settings:
   - **Build command**: (leer lassen)
   - **Publish directory**: `.` (root)
6. **Deploy site** klicken

### Schritt 4: FERTIG! 🎉
Deine App ist jetzt LIVE!
- URL: `https://banklernapp-123.netlify.app`
- Teile den Link mit deinem Sohn!

## 💡 Nutzung

### Für den Schüler:
1. Link öffnen
2. **Ohne Login starten** (lokal gespeichert)
3. Lernfelder durcharbeiten
4. Quizzes machen
5. Tests simulieren
6. Fortschritt sichtbar!

### Für Verbesserungen:

- Fertig! ✨

| LF | Titel | Wichtigkeit | Lerntag |
|---|---|---|---|
| 1 | Eigene Rolle - Rechtliches | 🔴 HIGH | Dienstag |

## 🎯 3-Tage Lernplan

**DIENSTAG (3h):**
- Lernfeld 1 + 2 Anfang
- Quiz + Merksätze

**MITTWOCH (3h):**
- Lernfeld 2 Fortsetzung + 3 + 4
- Tests & Schwachstellen erkennen

**DONNERSTAG (3h):**
- Lernfeld 5
- Volltest simulieren
- Schwache Punkte üben

**FREITAG:**
- 30 Min Wiederholung
- Zur Prüfung gehen
- **BESTEHEN! 🏆**
